## 🤔 首先，你需要反思

在开始保研之前，建议你真的想清楚**未来想干什么**。

### 1️⃣ 学位选择
目前主要的学位类型有：
- 学术型硕士（学硕）
- 专业型硕士（专硕）
- 直博
- 工程硕博

### 2️⃣ Title、方向、导师的trade off
**个人优先级排序**（仅供参考）：
```
热门方向 > 导师实力/学术圈影响力 > 地域（上海、杭州、大湾区、北京.etc）> 导师风格 > 学位类型
```

**学位喜好度排序**：
```
学硕 > 直博(港新) > 专硕 > 其他直博 > 工程硕博
```

**申请难度排序**：
```
直博（港新、USA）> 学硕 > 其他
```

### 3️⃣ 未来职业方向
早日想好你的未来：
- 继续从事学术研究？
- 前往央企/国企？
- 进入大厂做算法？
- 计划留学？

**个人经历**：我个人想早日接触工业界。就Foundation Models而言，我申请过Seed的实习，但由于没有顶会一作，最终被拒。这让我意识到，**大厂热门方向的竞争远超想象**。

## 🔤 保研常见术语解析

| 术语 | 含义 | 说明 |
|------|------|------|
| `rk` / `rank` | 排名 | 你在本校专业的排名 |
| `title` | 学校水平/牌子 | 985/211等学校标签 |
| `wl` | Waiting List | 候补名单 |
| `oq` | Over Qualified | 申请者实力远超院校水平，部分院校为防止被"海会"会筛掉这类申请者 |
| `bar` | 门槛 | 最低录取要求 |
| `强com` | 强committee | 招生办/审核组权力极大，导师基本无话语权（难以"捞人"） |
| `弱com` | 弱committee | 导师有较大权力 |

![rank](/image/blog/rank.png)

**常见专业代码**：
- `0812` - 计算机科学与技术
- `0835` - 软件工程
- `0839` - 网络安全
- `0854` - 电子信息专硕
  - `085404` - 计算机技术
  - `085405` - 软件工程

## 🎯 什么真正重要？（优先级排序）

```
Connection（人脉/导师推荐）
    ↓
学校层次
    ↓
Rank（排名）
    ↓
科研成果
    ↓
AI/算法竞赛
    ↓
英语成绩（CET-6，425/480已足够）
    ↓
其他竞赛（主要是本校保研加分）
```

### 关键建议

#### 1. 主动出击 - 提前套磁
- 无论学校是强Committee还是弱Committee，都要**提前联系老师**
- 你没什么可损失的，**不联系就一定没有机会**
- 提前建立联系 = 增加选择权

#### 2. 关于学校层次
- ⚠️ **哈工大基本别想**（非常难）
- 一些学校明确限制只要985院校
- 去年甚至有些学校差点只要清北

#### 3. 关于排名要求
很多学校入营有明确的排名卡线：
- 211 Rank 1
- Top 5% ~ 10%

💡 **夏令营的bar最高，进不了也不用灰心** — 很多学校预推免时会放宽标准。

#### 4. 科研的重要性
对于GPA不突出的同学：
- **科研就是你最大的筹码**
- 有科研经历 = 具备"即插即用"的能力
- 很多老师看重这一点，也有老师看学生潜力

**面试流程**（无弱Committee情况）：
```
师兄面 → 导师面 → 学院面 → 导师面 → 获得Offer
```

## ⚠️ 降低期待值

如果你一心想去**热门AI方向**，我得泼盆冷水：
> **清北华五热门AI方向，你大概率一个都去不了。**

**现状分析**：
- 那里面是"神仙打架"
- 一篇顶会现在**完全不够看**
- 往往需要多篇顶会 + 强背景

**现实情况**：
- 为了Title去这些学校：
  - 要么Rank 1去套磁北大软微
  - 要么去华五读直博
  - 要么选比较"坑"的方向/导师
- 大多数人的上限 = **中国AI强势学校（Top 9-10）的热门方向**
- 更多人的结果 = **末位9校或次9校**

💭 **现实**：大多数申请者都没有科研经历，GPA没什么差别。

## 💡 个人建议

### 建立自己的评价体系
推荐学习参考：
- 🎙️ **Jiayi Weng**（OpenAI Infra Core Member）的Podcast
- 思考什么真正对你的未来职业发展有帮助

### 两条路径选择

#### 📈 路径A：稳定卷学习
- **核心**：卷GPA是最稳的方法
- 大学三年间拿个竞赛/奖项加分
- Rank足够高 → 保研资格稳定 → 最次选末9或次9
- ✅ **优点**：稳定、压力小
- ❌ **缺点**：收获相对单一

#### 🚀 路径B：通过竞赛/科研突破
- **适合**：不喜欢卷GPA或对AI特别感兴趣的同学
- 获得保研资格 → 最终大家GPA差距都不大
- ✅ **优点**：获得全方位能���发展，而非冷冰冰的GPA数字
- ❌ **缺点**：需要投入更多时间和精力

## 🔬 如何增加科研经历？

这是我被问最多的问题。科研分为**本校**和**外校**两种。

### 📍 本校科研

**怎么找老师**：
- 查看**理学院、CS、控制、海空等学院**的老师
- 通过**Google Scholar**查看老师近期文章质量
- 咨询**往届学长学姐和同班同学**的评价

### 🌍 外校科研（RA / Visiting Student）

**形式**：On-site 或 Remote

**如何联系**：
- 访问清北等高校老师的主页
- 发送邮件表达科研兴趣
- 附上你的简历和科研经历

**小建议**：
```
第一阶段（本校）：
  ✓ 完成第一段科研
  ✓ 发表论文并获得本校加分
  ✓ 以此经历去套外校老师

第二阶段（外校）：
  ✓ 选择喜欢的方向
  ✓ 一般Remote需要较强实力
  ✓ 在外校深化科研
```

## 📝 机试、笔试、面试、专业课指南

> 以下主要针对AI相关院校

### ✏️ 笔试
- 🔵 **南科大统计学院** - 需要笔试

### 💻 机试（算法编程）
**需要机试的学校**：北航、厦大、天大等

**准备资源**：
- LeetCode Hot 100
- AcWing 算法基础课
- 剑指Offer

### 👔 面试
**特点**：根据你的简历**量身定做**

**实战例子**：
- 上科大面试：逐项深入询问简历上的所有科研细节
- 甚至拷打Diffusion的数学推导

### 📚 专业课（可选）
**AI相关常见考察**：
- 概率论
- 线性代数
- 机器学习
- 408（计算机基础）

**个人策略**：
- 我**没有深入准备**专业课（概率论、线代、408）
- 只准备了**AI相关八股文和常见算法**
- 特意选择**不太问专业课的学校**
- 结果：虽然某校对概率论、线代问得很深入，但最终也拿到了Offer

💡 **建议**：了解目标学校的出题风格，有针对性地准备。
